# GitHub Copilot

AI-powered code completion tool by GitHub (Microsoft). One of the earliest and most commercially successful [[Foundation Models]] applications in production.

## Key Facts

- Crossed $100M annual recurring revenue within 2 years of launch.
- Commonly used as a plug-in in VSCode.
- One of the earliest proofs that [[Foundation Models]] could power viable commercial products.

## Relevance

A flagship example of the [[Coding with AI]] use case. Demonstrates the commercial viability of AI engineering products.
